## 表1固定列-添加tableLayout:fixed彻底消除间隙 (15:29)

### 问题根源（关键发现）：
表格使用默认的 `table-layout: auto`，浏览器会**忽略** th/td 上设置的 `width` 值，根据内容自动调整列宽。导致：
- # 列 `width: 38` 被浏览器缩小到约 24px（只容纳 "#" 一个字符）
- 村社列 `left: 38` 按固定值定位
- 两者之间出现 ~14px 的透明间隙

### 修复方案（3处改动）：
1. **表格**：添加 `tableLayout: 'fixed'` — 强制所有 width 值被严格遵守，不因内容自动调整
2. **# 列**：`width: 44` + `boxSizing: 'border-box'`，`left: 0`
3. **村社列**：`left: 45`（= 44 width + 1 borderRight），去掉多余 boxShadow
