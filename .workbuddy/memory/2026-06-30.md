## 修复: 村社数据看板白屏问题 (YuzhiSyncDimension.tsx)

**问题**: 页面完全白屏，React root 无子元素 (rootChildren: 0)，Vite build 正常通过但运行时崩溃。

**根因**: `availableMonths` useMemo 回调中引用了 `allVillages`，但 `allVillages` 的定义在 useMemo 之后，导致 JavaScript TDZ (暂时性死区) ReferenceError。

```tsx
// 问题代码（错误顺序）：
const availableMonths = useMemo(() => {
    for (const r of allVillages) ...  // allVillages 尚未定义！
}, [allVillages])

const allVillages = VILLAGE_ROWS  // 定义在后面，useMemo 回调中访问时为 TDZ

// 修复：交换顺序
const allVillages = VILLAGE_ROWS
const availableMonths = useMemo(() => { ... }, [allVillages])
```

**调试方法**: 使用 agent-browser 打开页面，检查 root 子元素数量，确认编译无法发现的运行时错误。
