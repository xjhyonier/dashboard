## 环比/同比数据需求规划

### 优先级与价值矩阵

| 优先级 | 指标 | 类型 | 价值说明 | 计算方式 |
|:------:|------|:----:|---------|---------|
| **P0** | 隐患新增环比 | 环比 | 立即发现隐患是否异常增加 | 本周 vs 上周、本月 vs 上月 |
| **P0** | 隐患整改率环比 | 环比 | 判断整改效率是否提升 | (本期整改/上期整改) - 1 |
| **P0** | 超期存量环比 | 环比 | 监控超期风险是否缓解 | 期末超期数变化 |
| **P1** | 重大隐患环比 | 环比 | 关注高危风险变化 | 重大隐患新增/存量变化 |
| **P1** | 整改完成环比 | 环比 | 评估工作进度 | 本期闭环数 vs 上期 |
| **P2** | 隐患新增同比 | 同比 | 与去年同期对比 | 本期 vs 去年同期 |
| **P2** | 整改率同比 | 同比 | 评估年度改善 | 本期整改率 vs 去年同期 |
| **P2** | 企业合规达标率同比 | 同比 | 长期改善评估 | 各维度达标率年度对比 |
| **P3** | 专家工作量同比 | 同比 | 评估专家效率变化 | 本期 vs 去年同期 |
| **P3** | 检查覆盖率同比 | 同比 | 评估覆盖是否提升 | 检查企业数变化 |

---

### P0 - 必须展示（核心预警指标）

#### 1. 隐患新增环比
```
┌─────────────────────────────────────────┐
│  本周新增隐患                              │
│  32 处              ↑12% vs 上周(28处)   │
│                          ↑ 环比增加      │
│  ⚠️ 连续2周增长，关注异常！               │
└─────────────────────────────────────────┘
```
- **数据**: 本期新增数、上期新增数、环比变化率
- **判断**: 连续2期增长 > 20% 触发预警
- **价值**: 及时发现隐患异常增加，快速响应

#### 2. 超期隐患存量环比
```
┌─────────────────────────────────────────┐
│  超期未整改                              │
│  8 处              ↓33% vs 上周(12处)  │
│                          ↓ 环比减少      │
│  ✅ 连续3周下降，整改成效显著            │
└─────────────────────────────────────────┘
```
- **数据**: 当前超期数、上期超期数、环比变化
- **判断**: 存量 > 10 或 连续增长 触发预警
- **价值**: 核心风险指标，超期=最高风险

#### 3. 整改率环比
```
┌─────────────────────────────────────────┐
│  本月整改率                              │
│  87%              ↑3pp vs 上月(84%)    │
│                          ↑ 提升中       │
│  ✅ 已达目标线(85%)                    │
└─────────────────────────────────────────┘
```
- **数据**: 本期整改率、上期整改率、环比变化(百分点)
- **判断**: < 80% 红色预警，> 90% 绿色达标
- **价值**: 反映整体治理效果

### P1 - 重要指标（工作评估）

#### 4. 重大隐患环比

- **数据**: 本期新增重大隐患数、上期数、环比变化
- **判断**: 任何增长都值得关注
- **价值**: 最高风险等级，需优先处理

#### 5. 整改完成数环比
```
┌─────────────────────────────────────────┐
│  本月整改完成                            │
│  45 处            ↑15% vs 上月(39处)   │
│                          ↑ 效率提升     │
└─────────────────────────────────────────┘
```
- **数据**: 本期闭环数、上期闭环数、环比变化
- **判断**: 下降时需分析原因
- **价值**: 反映整改效率

### 数据展示格式规范

#### 环比数据格式
```
[当前值] [变化方向图标] [变化率] vs 上期([上期值])

示例：
32 处  ↑12% vs 上周(28处)
87%   ↑3pp vs 上月(84%)
```

#### 同比数据格式
```
[当前值] [变化方向图标] [变化率] vs 去年同期([去年值])

示例：
128 处  ↓8% vs 去年同月(139处)
86%    ↑6pp vs 去年同期(80%)
```

#### 预警状态标记
| 状态 | 标记 | 触发条件 |
|------|------|---------|
| 正常 | ✅ | 趋势向好或稳定 |
| 关注 | ⚠️ | 连续2期同向变化但未超阈值 |
| 预警 | 🔴 | 连续2期超阈值变化 |
| 达标 | 🟢 | 达到预设目标线 |

## 背景（原有内容）

当前应急管理看板有 5 个维度：
- **责任维度** - 组织与人员配置、安全制度建立
- **状态维度** - 企业状态路径图（开通→采集→授权→风险识别→整改闭环）
- **隐患维度** - 隐患详情列表（发现→整改→验收→闭环）
- **行业维度** - 按行业分类查看
- **专项维度** - 按工作组/任务计划查看

这些维度都是"**当前状态**"或"**当期数据**"视角，缺少**时间维度**的对比分析。

## 设计目标

新增一个 **趋势维度（Trend Dimension）**，用于展示关键指标随时间的变化趋势，帮助管理者：
1. 发现异常波动（隐患突增、整改率下降）
2. 评估治理效果（环比/同比改善）
3. 预测风险（超期隐患预警）

### 核心指标卡片说明

| 指标 | 计算方式 | 展示内容 | 预警规则 |
|------|---------|---------|---------|
| **新增隐患** | 本期新发现隐患数 | 当前值、环比变化（↑↓%）、上期值 | 连续2期增长>20%标红 |
| **整改完成** | 本期闭环隐患数 | 当前值、环比变化、完成率 | 连续2期下降标红 |
| **超期隐患** | 当前未闭环且超期数 | 当前值、环比变化、上期值 | 超期数>10 或 增长标红 |
| **整改率** | 已整改/应整改×100% | 当前率、环比变化、上期率 | <80% 标红 |

## 数据需求

### 现有数据支持

数据库已有这些数据支持趋势计算：

1. **Hazard** - 隐患记录
   - `discovered_at` - 发现时间
   - `status` - 状态（可判断闭环时间）
   - `level` - 等级（重大/一般）
   - `deadline` - 整改期限
   - `closed_at` - 闭环时间

2. **ExpertWorkload** - 专家工作量
   - `week_key` / `month_key` - 时间维度
   - `work_type` - 工作类型
   - `count` - 工作量

3. **EnterpriseDimensions** - 企业维度数据
   - `hazard_self` / `hazard_monitor` / `hazard_major` - 各类隐患数
   - 各维度完成率

## 视觉建议

### 配色方案

| 指标 | 正向颜色 | 负向颜色 | 说明 |
|------|---------|---------|------|
| 新增隐患 | `#6B7280`（灰色） | `#DC2626`（红色，增长时） | 灰色为默认，增长时变红 |
| 整改完成 | `#059669`（绿色） | `#DC2626`（红色，下降时） | 绿色表示良好 |

| 整改率 | `#059669`（绿色） | `#DC2626`（红色） | 高=绿，低=红 |

### 图表样式

- 折线图线条宽度：2px
- 数据点圆点大小：6px
- Hover 时放大到 8px + 显示 tooltip
- 图例位于图表上方或右侧
- 支持触控滑动查看（移动端友好）

## 与现有维度的联动

### 从其他维度跳转趋势

| 源维度 | 跳转参数 | 趋势页面响应 |
|-------|---------|-------------|
| 责任维度 | `dimension=duty` | 显示"职责制度"完成率趋势 |
| 隐患维度 | `level=major` | 只显示重大隐患趋势 |
| 状态维度 | `teamName=xxx` | 只显示该工作组趋势 |

### 趋势维度跳转其他维度

| 趋势指标 | 跳转目标 | 参数 |
|---------|---------|------|
| 新增隐患 | 隐患维度 | `status=pending&date=2024-03` |
| 超期隐患 | 隐患维度 | `status=overdue` |
| 某企业异常 | 企业状态 | `enterprise=xxx` |

## 风险与注意事项

1. **数据量问题** - 月度/年度趋势需要聚合计算，确保性能
2. **时间范围限制** - 建议最多展示 12 个月，避免渲染卡顿
3. **空数据处理** - 无数据的月份显示断点或虚线
4. **同比数据** - 如果有去年数据，建议增加同比对比
